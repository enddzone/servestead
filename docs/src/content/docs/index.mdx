Servestead turns a fresh Ubuntu VPS into a hardened host for private application stacks. It can provision a server, create an administrative user, harden SSH and the operating system, configure Docker networking, deploy Pangolin-backed ingress, and add observability.

## The Fastest Beginner Path

1. [Check the prerequisites](/getting-started/prerequisites/).
2. [Build the CLI](/getting-started/build/).
3. [Run guided setup on an existing VPS](/getting-started/existing-vps/).
4. [Point DNS and deploy the proxy](/guides/dns-and-proxy/).
5. [Add your first application stack](/guides/add-stack/).

:::caution[Run this on a disposable server first]
Servestead is intentionally powerful. It changes SSH access, firewall policy, Docker networking, reverse-proxy configuration, and server packages. Practice on a fresh VPS before using it on a machine with data you care about.
:::
